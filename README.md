# Oneil Hermez Portfolio Site

A static GitHub Pages-ready portfolio for **Oneil Hermez — Design, Modeling & Systems Thinking**.

## What is included

- `index.html` — main one-page portfolio
- `style.css` — responsive dark technical theme
- `script.js` — mobile navigation, header state, reveal animations
- `assets/images/` — portfolio images and video poster frames
- `assets/videos/` — uploaded Blender process and walkthrough clips
- `assets/favicon.svg` — simple site icon

## Portfolio focus

The site is written around:

- mechanical 3D modeling
- energy and nuclear-system-inspired design
- concept-to-final design thinking
- optimization and problem solving
- Blender and Photoshop presentation work
- physical builds and interactive asset work as supporting sections

## How to publish on GitHub Pages

1. Create a GitHub repository for the site.
2. Upload all files in this folder to the repository root.
3. In the repository settings, enable GitHub Pages from the main branch and root folder.
4. Replace or add images/videos as your portfolio grows.

## Things to customize later

- Replace the email in `index.html` if you want a different public contact address.
- Add links to LinkedIn, ArtStation, YouTube, or a resume when ready.
- Add more process images: wireframes, UV maps, blockouts, before/after Photoshop edits, and final renders.
- Consider hosting larger videos externally later if the repository becomes too large.
